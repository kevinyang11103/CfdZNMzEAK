# 医疗云管理系统 java1042

## 项目概述

医疗云管理系统是一款基于前后端分离架构的全新管理系统。系统以RuoYi-Vue为框架，提供了科室管理、医生排班、患者预约、数据分析等丰富功能。代码结构规范，注释详尽，便于学习和二次开发。

## 技术栈

- 后端：SpringBoot
- 前端：Vue3
- UI框架：Element Plus
- 数据库：MySQL

## 功能模块

### 系统角色

#### 管理员功能

- 新增、编辑、删除科室信息
- 医生信息录入、职称管理、账号分配
- 可视化排班表，智能冲突提醒
- 查看所有预约记录，支持手动调整
- 统计就诊量、医生接诊率、患者满意度

#### 医生端功能

- 查看个人排班表，申请调班
- 管理患者预约，调整接诊状态

#### 患者端功能

- 在线预约，查看预约记录，取消或改期
- 对医生服务进行评分

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：Node.js 12.0+
